# 🚗 Vehicle and Fuel Theft Detection System

## 📌 Description
This project is designed to detect and prevent vehicle and fuel theft by analyzing usage patterns and identifying suspicious activities. The system uses data analysis and anomaly detection techniques to enhance security and monitoring.

---

## 🎯 Objectives
- Detect unauthorized vehicle usage
- Identify abnormal fuel consumption
- Provide alerts for suspicious activities
- Improve vehicle security using smart monitoring

---

## ⚙️ Features
- 📊 Data analysis of vehicle usage
- ⛽ Fuel consumption monitoring
- 🚨 Theft detection alerts
- 📈 Visualization of data (graphs/charts)
- 🔐 Secure system access

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib / Seaborn (for visualization)
- Machine Learning (optional)
- CSV Dataset
